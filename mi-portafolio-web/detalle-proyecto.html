Detalle del proyecto el juego del laberinto

EL laberinto del preso: en busca de la libertar

Este proyecto consiste en un videojuego de texto donde se recorre un laberinto con caracteres ASCII. El caracter # representa una pared, este . un pasillo y el caracter p el personaje, asi mismo para recorrrer el laberinto se usaran las flechas del teclado ↑ ↓ ← →.

Usamos varios conceptos importantes para llegar al resultado de este videojuego como:

  • Bucles
  • Listas
  • Tuplas
  • Mapas
  • Matrices
  • Encapsulamiento

Cada concepto es aplicado para el correcto funcionamiento del laberitno.
